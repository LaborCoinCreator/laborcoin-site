What It Is

LaborCoin is a system for building a shared treasury that can support workers during strikes and collective action.

The strategy is simple: contribute regularly, grow a common fund, and deploy it when workers need support.

How It Works

  1. Use POL to buy LABR through the bonding curve
  2. Register to receive LABRV voting rights
  3. Vote on how funds are allocated
  4. DAO executes support for workers

LaborCoin System Overview

  • Total Supply: 1,000,000,000 LABR
  • Initial Supply: 100,000,000 LABR
  • Bonding curve pricing
  • 12-hour transaction cooldown
  • Exchange Wallet Limit: 10,000 LABR
  • Exchange Transaction Limit: 5,000 LABR
  • 10% total tax on sales:
  • 5% sales tax to DAO Treasury
  • 5% of each sale is distributed among eligible LABR holders
  • 10% of each LABR purchase goes to DAO Treasury

All LaborCoin smart contracts are publicly verifiable on Polygonscan and operate using immutable, transparent rules. Following final launch, ownership has been renounced or permanently delegated where applicable, and governance operates through the LaborCoin DAO.

Trust & Safety

  • On-chain governance with one verified participant per LABRV
  • Non-transferable voting tokens prevent accumulation of control
  • Governance proposals are restricted to treasury transfers only
  • Deterministic USD-denominated bonding curve pricing with Chainlink POL/USD conversion
  • Public, verifiable smart contracts on Polygon

The system is designed to minimize trust requirements and maximize transparency.

Onboarding Guide FAQ Disclaimer

Welcome to LaborCoin

LaborCoin uses four separate credentials that serve different purposes.

Human Passport (formerly Gitcoin Passport)
Identity verification used to access LaborCoin systems.

POL
Polygon network token used to purchase LABR through the exchange.

LABR
Transferable token used within the LaborCoin ecosystem.

LABRV
Non-transferable governance voting rights received through DAO registration.

Before using LaborCoin, users must complete Human Passport verification.

Human Passport Onboarding Guide

How LaborCoin Works

1. Buy LABR

Hold POL on Polygon.

Complete Passport verification.

DAO registration is NOT required to buy LABR.

LABR is purchased using POL. Make sure your wallet contains POL on Polygon before using the exchange.

2. Join the DAO

Hold at least 1 LABR.

Register to receive LABRV voting rights.

3. Participate in Governance

Create proposals.

Vote on treasury allocations.

Execute approved proposals.
